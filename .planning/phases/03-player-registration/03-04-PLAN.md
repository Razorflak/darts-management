**`+page.server.ts`:** Extend the existing dashboard load to return open events with registration state.

Add a query that fetches events with `status = 'ready'` and their tournaments with per-tournament registration state for the current player. Group tournaments under their event:

```sql
SELECT
  e.id AS event_id, e.name AS event_name, e.starts_at, e.ends_at,
  e.location, ent.name AS entity_name,
  t.id AS tournament_id, t.name AS tournament_name, t.category, t.check_in_required,
  COUNT(r.id)::int AS registration_count,
  (r_me.id IS NOT NULL) AS is_registered
FROM event e
JOIN entity ent ON ent.id = e.entity_id
JOIN tournament t ON t.event_id = e.id
LEFT JOIN tournament_registration r ON r.tournament_id = t.id
LEFT JOIN tournament_registration r_me
  ON r_me.tournament_id = t.id AND r_me.player_id = $currentPlayerId
WHERE e.status = 'ready'
GROUP BY e.id, ent.name, t.id, r_me.id
ORDER BY e.starts_at, e.name, t.name
```

`$currentPlayerId = locals.player?.id ?? null`. When null, `r_me.id IS NOT NULL` is always false.

Validate rows with `z.array(...)` using an inline schema for the flat row, then group by event_id in TypeScript into an array of `{ event: { id, name, starts_at, ends_at, location, entity_name }, tournaments: TournamentWithRegistration[] }`.

Return `{ openEvents }` from the load function (alongside existing return values if any — do not break existing dashboard data).

   ```sql
   SELECT t.id, t.name, t.category, t.check_in_required,
          e.id AS event_id, e.name AS event_name, e.status
   FROM tournament t JOIN event e ON e.id = t.event_id
   WHERE t.id = $tournamentId
   ```
   Validate with inline z.object schema. If not found: `error(404, 'Tournoi introuvable')`.
3. Fetch the roster:
   ```sql
   SELECT r.id AS registration_id, r.player_id, p.first_name, p.last_name,
          p.licence_no, r.checked_in, r.registered_at
   FROM tournament_registration r
   JOIN player p ON p.id = r.player_id
   WHERE r.tournament_id = $tournamentId
   ORDER BY p.last_name, p.first_name
   ```
   Validate with `z.array(RosterEntrySchema).parse()`.
