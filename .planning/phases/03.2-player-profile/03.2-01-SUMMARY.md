# Phase 03.2 Plan 01: Hooks Lookup-Only + Layout Player Propagation Summary

**Removed auto-create player from hooks.server.ts so locals.player = null when no profile exists, and exposed player in the (app) layout so all child routes receive data.player without additional DB queries.**
